bitvisitor.com Python Bot Build Script
======================================

Description
===========
This script essentially builds all the Python packages needed for `pybitvisitor.py` to execute properly.

Build
=====
		sudo ./build

Usage
=====
		python ./pyvisitor.py -a "your bitcoin wallet address here"
